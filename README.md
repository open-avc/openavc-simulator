# OpenAVC Simulator

**This project has been moved into the main OpenAVC repository.**

The simulator now lives at [`openavc/simulator/`](https://github.com/open-avc/openavc/tree/main/simulator) in the main repo, with the UI at [`openavc/web/simulator/`](https://github.com/open-avc/openavc/tree/main/web/simulator). It is included automatically when you install OpenAVC.

For documentation:
- **Using the simulator:** [docs/simulator.md](https://github.com/open-avc/openavc/blob/main/docs/simulator.md)
- **Writing simulators for drivers:** [writing-simulators.md](https://github.com/open-avc/openavc-drivers/blob/main/docs/writing-simulators.md)
